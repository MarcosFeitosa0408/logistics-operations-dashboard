## 🚀 Sobre o Projeto

Solução desenvolvida para análise histórica de KPIs logísticos com foco em tomada de decisão operacional — unindo identidade visual forte, experiência analítica fluida e narrativa clara de dados.

> *"O dashboard conseguiu unir identidade visual forte, experiência analítica fluida e KPIs realmente relevantes para contexto operacional e logístico — principalmente na análise de SLA, performance por rota e acompanhamento histórico."*
> — Marinize Santana, Engenheira de Dados

## 🛠️ Stack Utilizada

- **Power BI** — Visualização e dashboards interativos
- **Python** — Processamento e automação de dados

- **AWS** — Infraestrutura e armazenamento
- **PySpark** — Processamento de grandes volumes de dados

---

## 🎯 Objetivo

Criar um painel corporativo com narrativa clara que permita:

- Acompanhamento em tempo real da performance logística
- Análise histórica de KPIs operacionais
- Suporte direto à tomada de decisão estratégica
- Monitoramento de SLA e performance por rota
